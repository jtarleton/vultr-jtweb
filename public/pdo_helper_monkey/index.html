PHP Helper Monkey

PHP Helper Monkey is a collection of simple scaffolding tools for PHP. Alternatively to the many code generation libraries and framework components available through composer, the PHP Helper Monkey is a simple online tool. There are no dependencies and there is nothing to install.

If PHP Helper Monkey has helped you, please consider supporting this site through a tiny donation. Thank you.

Instructions

  2. Copy and paste Helper Monkey's output to use in your awesome project. That's it.
